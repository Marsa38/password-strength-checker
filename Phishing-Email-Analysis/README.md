# Phishing Email Analysis Project

## Overview
This project demonstrates basic skills in cybersecurity, focusing on phishing email investigation. 
The goal is to identify suspicious indicators, analyze headers, and verify URLs for security threats.

## Tools Used
- VirusTotal
- MXToolbox (for email header analysis)
- Text editor (for email examples)

## Steps
1. Analyzed email content for signs of phishing: urgency, threats, and suspicious links.
2. Verified URLs using VirusTotal to check for malicious classification.
3. (Optional) Analyzed email headers using MXToolbox to detect spoofing attempts.

## Results
- Identified phishing indicators in example email.
- Verified URL as malicious using VirusTotal.
- Documented all findings in notes.md and screenshots folder.

## Conclusion
This project shows the ability to apply cybersecurity fundamentals to practical phishing detection scenarios.
